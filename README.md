# Mr. Plow

Mr. Plow is a first-person snow removal simulator built with the Godot Engine.

Players clear snow from residential properties, earn money for completed work, and unlock increasingly powerful equipment such as upgraded snow shovels, snow blowers, and snow plows.

This project is currently in active development.

---

## Features

Current features include:

- First-person player controller
- Mouse look
- Snow removal mechanics
- Upgrade system
- Multiple snow removal tools
- Simple property environment
- HUD with player information
- Procedurally generated prototype assets

Planned features include:

- Multiple properties
- Economy and shop system
- Equipment progression
- Vehicles
- Dynamic weather
- Save/load system
- Steam integration

---

## Requirements

Before running the project, make sure the following software is installed:

- Godot Engine 4.7 or newer
- Git
- Optional:
  - VS Code
  - VSCodium

---

## Getting Started

Clone the repository:

```bash
git clone https://github.com/Akkocak07/mr-plow.git
```

Navigate into the project directory:

```bash
cd mr-plow
```

Open the project using the Godot Editor:

```
project.godot
```

Press **F5** to launch the game.

---

## Installation

### Linux

Install Godot using Flatpak:

```bash
flatpak install flathub org.godotengine.Godot
```

Alternatively, download the latest release directly from:

https://godotengine.org/download

---

### Windows

1. Download Godot from

https://godotengine.org/download

2. Extract the archive.

3. Launch the Godot executable.

4. Open the project's `project.godot` file.

## Controls

| Input | Action |
|--------|--------|
| W | Move Forward |
| A | Move Left |
| S | Move Backward |
| D | Move Right |
| Mouse | Look Around |
| Left Mouse Button | Remove Snow |
| Space | Remove Snow |
| U | Purchase Upgrade |
| R | Regenerate Snow |
| Escape | Release Mouse Cursor |
